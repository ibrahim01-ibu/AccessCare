# Shared Resources

This directory contains shared documentation and configuration for the AccessCare project.

## Structure

- `api-contracts.md` - Complete API specification for all services
- `design-flows/` - User flow diagrams and architecture documentation
- `config/` - Shared configuration templates and schemas

## Usage

All team members should reference this directory for:
- API contract definitions before implementing features
- Design patterns and user flows
- Configuration templates

## Contribute

When adding new features:
1. Update API contracts in `api-contracts.md`
2. Add design flow diagrams to `design-flows/`
3. Update configuration templates if needed
